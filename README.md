## 📖 Sobre o Projeto
O **TravelGram** é uma página web estática desenvolvida em **HTML e CSS**, que simula uma rede social voltada para viagens.  
O objetivo é praticar conceitos de front-end e criar uma interface visual para exibir perfis e fotos de destinos turísticos.
